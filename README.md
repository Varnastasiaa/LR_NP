# LR_NP

Лабораторна робота з обробки векторів `double` на C++.

## Що робить програма

Програма:
- запитує кількість елементів `n`;
- генерує вектор `v` з `n` випадкових чисел у діапазоні `[1; 100]`;
- записує `v` у файл `a.txt`;
- запитує розмір групи `k`;
- обчислює суми по групах (крок `k`) і записує їх у `b.txt`;
- формує:
1. `w1` - сортування `v` за зростанням;
2. `w2` - сортування `v` за спаданням;
3. `s2` - сума всіх групових сум;
4. `s3` - сума унікальних елементів `v`;
5. `max group` - індекс групи з максимальною сумою;
6. `copy` - кількість дублікатів у початковому векторі;
7. `countA` - кількість елементів у `v`;
8. `countB` - кількість групових сум.
- зберігає підсумок у `result.json`.

## Структура файлів

- `NP.cpp` - `main()`, введення даних і виклики обчислень.
- `NP.h` - реалізація допоміжних функцій.
- `a.txt` - згенерований вектор.
- `b.txt` - суми по групах.
- `result.json` - фінальний структурований результат.

## Збірка

```bash
g++ -std=c++17 NP.cpp -o NP
```

## Запуск

```bash
./NP
```

Під час запуску потрібно ввести:
1. `n` - кількість елементів (має бути `> 0`);
2. `k` - розмір групи (має бути `> 0`).

Якщо ввести некоректні значення, програма завершується з повідомленням про помилку.

## Приклад результату

Файл `result.json` містить поля:
- `w1`, `w2`
- `s2`, `s3`
- `max group`
- `copy`
- `countA`, `countB`

Для прикладу поточний `result.json` у репозиторії містить тестовий запуск для `countA = 10` та `countB = 5`.
